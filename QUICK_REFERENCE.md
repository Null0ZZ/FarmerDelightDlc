## 核心操作

### 5个基本步骤

| 步骤 | 操作 | 快捷键 |
|------|------|--------|
| 1️⃣ | 选择模组 | 左侧列表点击 |
| 2️⃣ | 切换模式 | 点击 **⭐ 编辑节点** |
| 3️⃣ | 创建节点 | 右侧 **➕ 创建根节点** |
| 4️⃣ | 编辑节点 | 点击节点 → 选择操作 |
| 5️⃣ | 保存更改 | **✅ 保存更改** |

## 节点编辑快速菜单

```
点击节点后出现的菜单：

📦 设置物品      → 选择关联的物品
📝 设置描述      → 编辑节点描述
🔗 设置父节点    → 建立依赖关系
🗑️ 删除节点      → 删除此节点
```

## 关键概念

### 什么是父节点？
```
父节点 (前置条件)
   ↑
   │ 依赖箭头
   │ (指向父节点)
   
子节点 (后续成就)
```

### 一对多关系
```
       [父节点]
        ↗ ↓ ↖
   [子1] [子2] [子3]
```

### 多对一关系
```
[前1] ←\
[前2] ← → [目标节点]
[前3] ←/
```

## 快捷操作

| 目的 | 步骤 |
|------|------|
| 创建节点 | 右侧 **➕** → Canvas 上显示新节点 |
| 选中节点 | 点击节点 → 高亮边框 + 右侧菜单 |
| 关联物品 | 点击 **📦** → 选择物品 → 点击关闭 |
| 建立关系 | 点击 **🔗** → 选择父节点 → 点击完成 |
| 编辑描述 | 点击 **📝** → 输入文字 → 失焦自动保存 |
| 删除节点 | 点击 **🗑️** → 确认 → 节点删除 |
| 保存所有 | 点击 **✅** → 数据保存到模组 |
| 返回分类 | 点击 **←** → 返回分类模式 |

## 常见错误和解决

| 问题 | 原因 | 解决方案 |
|------|------|--------|
| 节点不显示 | 未创建节点 | 点击 **➕ 创建根节点** |
| 连线不显示 | 节点间无关系 | 点击 **🔗 设置父节点** |
| 无法设置父节点 | 会形成循环 | 系统自动阻止，选择其他节点 |
| 数据丢失 | 浏览器缓存清除 | 使用 JSON 导入恢复数据 |
| 页面白屏 | JavaScript 错误 | 按 F12 查看控制台，刷新页面 |
